# ckd-detection-ml
Binary classification for Chronic Kidney Disease detection using ensemble ML methods | Accuracy: 98.75% | ROC-AUC: 1.0

Chronic Kidney Disease Detection using ML
Binary classification system to detect CKD using 5 ML algorithms 
(Logistic Regression, KNN, SVM, Decision Tree, Random Forest) 
+ ensemble voting.

Dataset:
UCI CKD Dataset (400 records × 25 features) 

Best Accuracy:
98.75% | ROC-AUC: 1.0

Tools: Python, Scikit-learn, Pandas, Matplotlib
